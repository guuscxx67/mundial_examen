Acerca del proyecto

Sistema del Mundial FIFA 2026

Aplicación web full-stack para simular, administrar y geolocalizar la Copa Mundial de la FIFA 2026, disputada en México, Estados Unidos y Canadá. El mismo frontend consume una API REST que puede respaldarse con MySQL, PostgreSQL o MongoDB; este equipo utiliza PostgreSQL.

Stack tecnológico

  • Frontend: HTML + CSS + JavaScript (vanilla) + Leaflet (mapas)
  • Backend: Node.js + Express (API REST)
  • Base de datos: PostgreSQL 16 (Docker)
  • POO: modelos con clase base y herencia

Módulos

    Requisitos cubiertos

    • Menú de confederaciones (UEFA, CONMEBOL, CONCACAF, CAF, AFC, OFC) con ficha emergente por selección.
    • Grupos con tabla PJ · PG · PE · PP · GF · GC · DG · PT.
    • Fase de dieciseisavos generada automáticamente con sedes asignadas.
    • Módulo de carga de resultados por fase: modificar un marcador de grupos re-siembra los dieciseisavos, y en eliminatorias el ganador avanza automáticamente ronda a ronda (con penales en caso de empate).
    • Estadios con ubicación en Google Maps, equipos, fechas, horarios y costo de boletos.
    • Geolocalización con mapa interactivo, ruta y compartir en redes sociales.
